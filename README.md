# Trider

Base for code gen (sandbox or replica) stuff

Replicate project or folder structure, using a hierarchical, tree-like directory or folder structure, that be generated via linux tree comand. 

Have a tree.txt file with dir structure.

# Run the script directly no lib or anything required

python3 ./main.py ./tree.txt -r ./

expn: python3 ./main_script_file.py ./input_tree_structure.txt -r ./root_dir
